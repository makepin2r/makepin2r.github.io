## 나의 풀이

### 코드

```javascript
function solution(rny_string) {
  return rny_string.replace(/m/g, "rn");
}
```

## 다른 사람 풀이

```javascript
function solution(rny_string) {
  var answer = "";
  return [...rny_string].map(e => (e == "m" ? "rn" : e)).join("");
}
```

문자열을 구조분해하여 `map`을 이용해 문자를 대체한 뒤 다시 join으로 합치는 방식의 풀이.
