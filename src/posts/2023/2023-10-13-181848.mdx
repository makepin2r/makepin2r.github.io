## 나의 풀이

### 코드

```javascript
function solution(n_str) {
  return Number(n_str);
}
```

## 다른 사람 풀이

```javascript
const solution = Number;
```

띠용... 함수에 Number 생성자 함수 바로 넣어버리기...

```javascript
function solution(n_str) {
  return +n_str;
}
```

`+` 단항 연산자를 이용해 숫자로 변환한 풀이이다.
