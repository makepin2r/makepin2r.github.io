## 나의 풀이

### 코드

```javascript
function solution(arr) {
  for (let i = 0; i < arr.length; ++i) {
    for (let j = 0; j < arr.length; ++j) {
      if (arr[i][j] != arr[j][i]) return 0;
    }
  }
  return 1;
}
```

## 다른 사람 풀이

```javascript
function solution(arr) {
  return arr.every((r, i) => r.every((_, j) => arr[i][j] === arr[j][i]))
    ? 1
    : 0;
}
```

`every` 메서드를 이용한 멋진 풀이!  
arr에 every 함수를 실행하면 2차원 배열의 모든 행마다 콜백 함수가 실행된다.  
콜백 함수의 두 번째 인자인 인덱스를 이용해 한 행 내의 모든 값을 테스트한다.

## 알게 된 것

### Array.prototype.every()

`every` 메서드는 배열 내 모든 요소가 인자로 전달된 함수를 true로 통과하는지 테스트한다.  
그러므로 every 함수 자체와 인자로 전달되는 콜백 함수 모두 boolean 값을 반환해야 한다.

**every()에 대한 설명**

> Array 인스턴스의 every() 메서드는 배열의 모든 요소가 제공된 함수로 구현된 테스트를 통과하는지 테스트합니다. 이 메서드는 불리언 값을 반환합니다. (출처: [MDN](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/every))

**every()의 인자 함수에 대한 설명**

> callbackFn : 배열의 각 요소에 대해 실행할 함수입니다. 요소가 테스트를 통과하면 참 값을 반환하고, 그렇지 않으면 거짓 값을 반환해야 합니다. 함수는 다음 인자를 사용하여 호출됩니다. (출처: [MDN](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/every))
