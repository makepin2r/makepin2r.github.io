## 나의 풀이
### 아이디어
filter 메서드를 이용해서 같은 인덱스에 있는 finished 배열의 값이 false인 원소만 리턴한다.
### 코드
```javascript
function solution(todo_list, finished) {
    return todo_list.filter((_, i) => !finished[i]);
}
```
