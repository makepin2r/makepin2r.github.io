## 나의 풀이
### 코드
```javascript
function solution(rsp) {
    return [...rsp].map(v => v === "2" ? 0 : v === "0" ? 5 : 2).join("");
}
```

## 다른 사람 풀이
```javascript
function solution(rsp) {
    let arr = {
        2: 0,
        0: 5,
        5: 2
    };
    var answer = [...rsp].map(v => arr[v]).join("");
    return answer;
}
```
객체를 활용한 풀이이다.  
각각 대응하는 고정된 값이 있으므로 객체의 key - value 조합을 활용해 풀 수 있다.  
앞으로 페어로 값을 대응할 수 있는 형태의 문제가 나오면 활용할 수 있을 것 같다.
