## 나의 풀이

### 코드

```javascript
function solution(my_string) {
  const answer = [];
  for (let i = 0; i < my_string.length; ++i) {
    const target = my_string.at(i);
    if (Number(target) == target) answer.push(Number(target));
  }
  return answer.sort((a, b) => a - b);
}
```

## 다른 사람 풀이

```javascript
function solution(my_string) {
  return my_string
    .match(/\d/g)
    .sort((a, b) => a - b)
    .map(n => Number(n));
}
```

정규식을 이용하면 반복문을 사용하지 않아도 된다!
