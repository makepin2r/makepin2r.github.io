## 나의 풀이

### 아이디어

배열의 인덱스를 5로 나눈 나머지가 0이면 5명 단위에서 가장 앞 원소가 나오므로, filter를 이용해 해당 원소들을 담은 배열을 반환한다.

### 코드

```javascript
function solution(names) {
  return names.filter((_, i) => i % 5 === 0);
}
```
